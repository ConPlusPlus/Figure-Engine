# Anvil

Anvil is Figure Engine's original level editor.

## Phase 1 goals

- Standalone `Anvil.exe`
- Top-down block map editing
- New / Open / Save / Save As for `.figmap`
- Block drawing
- Selection and movement
- Delete selection
- Properties and output panels
