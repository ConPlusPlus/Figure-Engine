# LevelEditor replacement folder

This is a drop-in starter Level Editor folder for `Figure Engine/Editor/LevelEditor/`.

## Included
- Visual Studio Win32 project
- Top-level editor window
- New/Open/Save/Save As for `.figmap`
- Block drawing tool
- Selection/move/delete
- Properties panel
- Output log
- Folder structure matching your existing editor layout

## Notes
- Entity/Texture tools are placeholders for future work.
- The map format is a simple text `.figmap`.
